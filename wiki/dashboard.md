# Live Dashboard

> All blocks below require the **Dataview** plugin (install from Settings → Community plugins → Browse → "Dataview"). Once installed, this page renders live the moment any frontmatter changes — no rebuild needed.
>
> The static BI dashboard lives at [`outputs/dashboard.html`](../outputs/dashboard.html) and is regenerated by the agent's Build operation. This page is its always-fresh, in-Obsidian counterpart.

## KPIs

```dataviewjs
const pages = dv.pages('"wiki"').where(p => p.type && p.type !== "dashboard" && p.type !== "index" && p.type !== "report");
const published = pages.where(p => p.state === "published");
const highConf = pages.where(p => p.confidence === "high");
const quality = pages.where(p => p.quality_score).map(p => p.quality_score);
const meanQ = quality.length ? Math.round(quality.array().reduce((a,b)=>a+b,0)/quality.length) : 0;

dv.table(["Metric", "Value"], [
  ["Total pages", pages.length],
  ["Published", published.length],
  ["High-confidence", highConf.length + " (" + Math.round(100*highConf.length/Math.max(pages.length,1)) + "%)"],
  ["Mean quality", meanQ],
]);
```

## Pages by type

```dataview
TABLE WITHOUT ID type AS "Type", length(rows) AS "Count"
FROM "wiki"
WHERE type AND type != "dashboard" AND type != "index" AND type != "report"
GROUP BY type
SORT length(rows) DESC
```

## Recently updated

```dataview
TABLE WITHOUT ID file.link AS "Page", type, confidence, quality_score AS "Q", updated
FROM "wiki"
WHERE type AND type != "dashboard" AND type != "index"
SORT updated DESC
LIMIT 15
```

## Needs work — low confidence

```dataview
TABLE WITHOUT ID file.link AS "Page", type, quality_score AS "Q", updated
FROM "wiki"
WHERE confidence = "low"
SORT quality_score ASC
```

## Stale (not touched in 30 days)

```dataviewjs
const cutoff = dv.date("today").minus(dv.duration("30 days"));
const stale = dv.pages('"wiki"')
  .where(p => p.updated && dv.date(p.updated) < cutoff)
  .where(p => p.type && p.type !== "dashboard" && p.type !== "index");
dv.table(["Page", "Type", "Updated"], stale.map(p => [p.file.link, p.type, p.updated]));
```

## Orphan candidates (empty `related`)

```dataview
TABLE WITHOUT ID file.link AS "Page", type, confidence
FROM "wiki"
WHERE type IN ["concept", "entity"] AND (!related OR length(related) = 0)
SORT file.name ASC
```

## Sources ingested

```dataview
TABLE WITHOUT ID file.link AS "Source", source_kind AS "Kind", source_date AS "Pub date", quality_score AS "Q", confidence
FROM "wiki/sources"
SORT source_date DESC
```
