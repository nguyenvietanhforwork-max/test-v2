# raw/news — IMMUTABLE

Drop incoming news clips here:
- Markdown captures from Obsidian Web Clipper
- HTML/markdown crawl exports
- Browser extension news clippings
- Pasted articles

**Rules (AGENTS.md §2):**
- NEVER edit, rewrite, or rename files in this folder.
- The agent processes them and produces wiki notes + reports.
- After a file is fully processed AND all reports + wiki + dashboard are updated, the agent moves it to `raw/old news/YYYY-MM-DD/` — never copies, never edits.

If processing fails for any file, it stays here.
