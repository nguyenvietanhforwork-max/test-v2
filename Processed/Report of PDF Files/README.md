# Report of PDF Files

Reports digesting research PDFs from `raw/PDF Files/`.

**Naming:** `NNN - YYYY-MM-DD.md`
**Template:** `_templates/report-pdf.md`

For every PDF processed:
- Publisher / author / document type
- Thesis (single paragraph)
- Key datapoints (table)
- Forecasts & targets (table)
- Strengths and weaknesses of argument
- Implications for our wiki coverage (which wiki notes were updated)
