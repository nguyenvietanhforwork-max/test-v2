# Report of Social Media Post

Reports of social signals from `raw/Social Media Post/`.

**Naming:** `NNN - YYYY-MM-DD.md`
**Template:** `_templates/report-social.md`

Sections:
- Sentiment snapshot
- Notable signals (per platform: X, FB, Telegram, LinkedIn, Threads, Reddit)
- Cross-platform themes
- Suspicious / low-quality flags

Each signal carries source attribution to the raw social post markdown and an engagement metric where available.
