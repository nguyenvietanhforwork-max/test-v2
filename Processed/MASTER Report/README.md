# MASTER Report

Cross-source daily synthesis that pulls from:
- `Processed/Report of news/`
- `Processed/Report of Social Media Post/`
- `Processed/Report of PDF Files/`

**Naming:** `NNN - YYYY-MM-DD.md`
**Template:** `_templates/report-master.md`

Sections:
- Narrative of the day (single dominant story)
- Cross-source trends (evidence from all three reports)
- Macro implications
- Sector rotation signals (table with strength rating)
- Emerging themes flagged for new wiki note creation
- Investment implications (NEVER recommendations)
- Watchlist updates (add/remove/hold)

This is the report that should be read first by a human reviewer each morning.
