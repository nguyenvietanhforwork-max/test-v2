   - is analyst-voice, forward-looking, drawing a consequence the source did not state outright

   - 18–35 words

4. `data_points` — 0–8 items. Each is a notable numeric, ratio, currency value, or named threshold extracted verbatim with a 6-word context. Empty list if none.
